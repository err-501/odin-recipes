# odin-recipes

A simple recipe website built as part of **The Odin Project** Foundations curriculum. This project focuses on practicing the fundamentals of HTML by creating a multi-page website with links between pages.

## Features

- Home page with links to multiple recipes
- Individual recipe pages
- Images for each recipe
- Recipe descriptions
- Ingredients listed with unordered lists
- Step-by-step cooking instructions using ordered lists
- Navigation links back to the home page

## Skills Demonstrated

By completing this project, I practiced:

- Writing semantic HTML
- Creating the basic HTML boilerplate
- Organizing files and folders
- Using headings, paragraphs, lists, and images
- Creating relative links between pages
- Building a simple multi-page website
- Structuring content for readability
